# PlayersEdge Group

A clean, professional corporate website for PlayersEdge Group - a performance marketing and affiliate media company operating in the legal U.S. online sports betting industry.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript**
- **Tailwind CSS** for styling
- **React** 18+

## Getting Started

### Development Server

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
npm run build
npm start
```

## Site Structure

- `/` - Home page with hero, company overview, and compliance information
- `/about` - About PlayersEdge Group
- `/how-it-works` - Explanation of the affiliate model
- `/responsible-gaming` - Responsible gaming resources and guidelines
- `/affiliate-disclosure` - FTC-compliant affiliate disclosure
- `/contact` - Business inquiry contact form

## Compliance Features

This site is designed for Meta's review process for online gambling advertising compliance:

- Clear disclosure that PlayersEdge Group is a media/affiliate company, not a sportsbook
- 21+ age requirements prominently displayed
- Legal jurisdiction restrictions clearly stated
- Responsible gaming information and resources
- Transparent affiliate relationship disclosures
- No direct gambling functionality (no bets accepted)

## Design

- Dark slate background with lime green accents
- Professional, corporate aesthetic
- Fully responsive design (mobile, tablet, desktop)
- Accessible navigation and footer on all pages
